## Project setup

```bash
$ pnpm install
```

### Compiles and hot-reloads for development

```bash
$ pnpm serve
```

### Compiles and minifies for production

```bash
$ pnpm build
```

### Preview for production

```bash
$ pnpm start
```

### Lints and fixes files

```bash
$ pnpm lint
```
